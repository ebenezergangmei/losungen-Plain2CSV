STEPS 1

How to create a separate Calendar in the Google Calendar

1. Open Google Calendar in your account.

2. On the left Panel, If you scroll down look for "Other calendars"

3. Click on the + button on the side

4. Click-> Create new calendar

5. Type in Name of the Calendar

   Type in the Descriptions of the Calendar

   Time zone (Which is usually by Default as local) - No need to touch

6. Click Create calendar









STEPS 2

How to Import CSV file into Google calendar

1. Open Google Calendar in you account.

2. On the left Panel, If you scroll down look for "Other calendars"

3. Click on the + button on the side

4. Click Import

5. Select the CSV file created earlier (from the local Computer)

6. "Add to calendar" - Click down button

7. Choose the Calendar you Created in the previous Steps

8. Click Import
